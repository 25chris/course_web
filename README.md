# Flutter Courses Website by 25Chris

Welcome to the repository for the **Flutter Courses Website**, a project crafted with care and expertise by [25Chris](https://github.com/25chris). This website is designed to offer comprehensive courses on Flutter and various other computer science topics, tailored for both beginners and advanced learners.

## About the Project

This project aims to provide an accessible and easy-to-navigate platform for learning computer science basics and Flutter framework development. Our design philosophy is simple and straightforward, ensuring that users can focus on learning without distractions.

### Features

- **User-friendly Interface**: Designed with simplicity and ease of use in mind.
- **Comprehensive Courses**: Covers topics from basic computer skills to advanced Flutter framework development.
- **Responsive Design**: Optimized for both web and mobile platforms.
- **Course Tracking**: Users can track their progress and resume courses where they left off.
- **Community Support**: Access to forums and discussions to enhance learning.

## Technology Stack

The website is built using the **Flutter** framework, leveraging the **Stacked CLI** for state management and project organization.

### Key Technologies Used:

- **Flutter**: A UI toolkit for crafting natively compiled applications for mobile, web, and desktop from a single codebase.
- **Stacked CLI**: A powerful tool for Flutter state management, enhancing the development workflow.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK: [Installation Guide](https://flutter.dev/docs/get-started/install)
- Dart SDK: Included with Flutter SDK
- Git: [Download and Install Git](https://git-scm.com/downloads)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/25chris/course_web.git
   ```
2. Navigate to the project directory
   ```sh
   cd course_web
   ```
3. Install dependencies
   ```sh
   flutter pub get
   ```
4. Run the app
   ```sh
   flutter run
   ```
